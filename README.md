# Big Data Analysis on Global E-Commerce Transactions using Spark


Dataset Source (Kaggle) 
Dataset to be used: Online Retail Dataset (UK E-Commerce) 
Link of the dataset: https://www.kaggle.com/datasets/lakshmi25npathi/online-retail-dataset 
This dataset contains 541,000+ rows of real transactional data. 



Project Objective :- 

Students will act as Data Engineers + Data Analysts for an e-commerce company. 
Their job is to: 
  1. Clean the raw transactional data 
  2. Process it using Spark 
  3. Extract meaningful business insights 
  4. Compare Spark vs Normal processing performance 
This simulates a real corporate analytics workflow.
